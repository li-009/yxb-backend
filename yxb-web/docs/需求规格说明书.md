# 鹦学伴 - 外语学习视频助手 核心页面设计

> 最后更新: 2024-12-30

# 一、设计说明

1. 设计依据：基于"外语学习视频播放器（含微信生态集成）需求规格说明书"，聚焦核心功能（视频播放、AI辅助学习、微信生态集成、个性化管理）与典型用户场景（碎片化学习、社群共学、精准练习）；

2. 适配平台：优先适配iOS/Android原生APP，关键布局兼容微信小程序（简化导航栏与交互）；

3. 设计原则：核心功能入口突出、操作流程简洁、学习场景沉浸式，符合外语学习用户"快速切换功能、精准获取辅助"的需求。

# 四、核心页面设计详情

## 页面1：首页（个性化推荐与功能入口页）

### 1. 核心功能
- 展示个性化学习资源推荐
- 提供核心功能快速入口（视频导入、AI跟读、单词复习）
- 呈现学习数据概览与待办任务
- 集成微信生态入口（分享、社群、扫码）

### 2. 布局结构
1. **导航栏（顶部）**
   - 左侧：产品Logo"鹦学伴"
   - 中间：搜索框（提示文字："搜索视频、单词、语法点"）
   - 右侧：消息通知图标（含未读红点提示）

2. **学习数据概览区（卡片式）**
   - 左侧大卡片：今日学习时长、连续打卡天数，附"今日打卡"按钮
   - 右侧小卡片：待复习单词数、未完成学习任务

3. **核心功能入口区（4宫格）**
   - 视频导入、AI跟读、单词复习、学习社群

4. **个性化推荐区**
   - 标题栏："为你推荐"+"换一批"按钮
   - 视频卡片：封面、标题、语种、难度、时长

5. **每日一句**
   - 英文原句 + 中文翻译

6. **底部导航栏**
   - 首页、资源库、+、学习、我的

## 页面2：视频播放页（核心学习页面）

### 1. 核心功能
- 视频播放与基础控制
- 字幕解析、切换与互动
- AI辅助学习功能（语法讲解、跟读评分、屏幕查词）
- 笔记、收藏、分享

### 2. 布局结构
1. **导航栏（顶部）**
   - 返回按钮、视频标题、收藏/分享按钮

2. **视频播放区**
   - 视频画面、字幕显示区

3. **核心控制栏**
   - 进度条、播放/暂停、单句循环、AB段复读、字幕设置、倍速

4. **AI学习工具栏（固定）**
   - 语法讲解、跟读评分、屏幕查词、添加笔记、分享

5. **字幕列表**
   - 可点击跳转、查词、语法分析

## 页面3：个人中心页

### 1. 核心功能
- 个人信息与学习数据展示
- 学习资产管理（笔记、收藏、单词本）
- 账号设置与微信关联

### 2. 布局结构
1. **个人信息区**：头像、用户名、语言水平、VIP标识
2. **学习数据统计区**：累计学习时长、掌握单词数、笔记数量
3. **核心功能入口**：我的笔记、我的收藏、单词本、学习报告、微信关联
4. **系统设置**：账号设置、播放设置、隐私设置、帮助中心

## 页面4：资源库页

### 1. 核心功能
- 外语学习视频资源分类展示
- 多维度筛选与精准搜索
- 资源收藏与快速学习

### 2. 布局结构
1. **导航栏**：返回按钮、搜索框、筛选按钮
2. **筛选面板**：语种、难度、资源类型、时长
3. **分类导航**：横向滚动标签
4. **视频网格列表**：封面、标题、语种、难度、热度、收藏按钮

# 五、页面关联逻辑

1. 首页 → 视频播放页：点击推荐资源卡片
2. 首页 → 资源库页：点击底部导航"资源库"
3. 首页 → 个人中心页：点击底部导航"我的"
4. 视频播放页 → 个人中心页：通过笔记/收藏查看
5. 资源库页 → 视频播放页：点击资源卡片
6. 个人中心页 → 视频播放页：点击笔记/收藏的资源
