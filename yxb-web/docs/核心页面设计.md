# 鹦学伴 - 外语学习视频助手 核心页面设计

## 一、设计说明

1. 设计依据：基于"外语学习视频播放器（含微信生态集成）需求规格说明书"，聚焦核心功能（视频播放、AI辅助学习、微信生态集成、个性化管理）与典型用户场景（碎片化学习、社群共学、精准练习）；

2. 适配平台：优先适配iOS/Android原生APP，关键布局兼容微信小程序（简化导航栏与交互）；

3. 设计原则：核心功能入口突出、操作流程简洁、学习场景沉浸式，符合外语学习用户"快速切换功能、精准获取辅助"的需求。

## 二、核心页面

### 页面1：首页
- 导航栏：Logo、搜索框、微信扫码、消息通知
- 学习数据概览区：今日学习时长、打卡天数、待复习单词、未完成任务
- 核心功能入口：视频导入、AI跟读、单词复习、微信社群
- 个性化推荐区：为你推荐、每日一句、热门影视片段、语法专题
- 底部导航栏：首页、资源库、+、学习记录、我的

### 页面2：视频播放页
- 视频播放区：双语字幕、倍速、亮度、音量
- 核心控制栏：播放/暂停、单句循环、AB段复读、字幕设置
- AI学习工具栏：语法讲解、跟读评分、屏幕查词、添加笔记
- 微信分享快捷入口

### 页面3：个人中心
- 个人信息区：头像、用户名、语言水平、VIP标识
- 学习数据统计：累计学习时长、掌握单词数、笔记数量
- 核心功能入口：我的笔记、我的收藏、单词本、学习报告、微信关联
- 系统设置：账号设置、播放设置、隐私设置、帮助中心

### 页面4：资源库
- 搜索框 + 筛选按钮
- 筛选面板：语种、难度、资源类型、时长
- 分类导航：全部、影视片段、商务英语、旅游外语等
- 资源列表：网格布局，含封面、标题、语种、难度、热度
